# 第一部分 大数据开发
## 数仓个人思考点

- 数据仓库为什么要分层 ？
  - 大多数情况，数据体系是依赖复杂，杂乱无章的。 不知不觉中，表与表之间会出现循环依赖的结构，这样的结构是混论的，低效的
  通过有效的数据组织和管理方法让我们的数据仓库更加有序，便于后续的维护和开发，应用。这就是分层的意义
